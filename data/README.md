# Data

This directory contains the GitHub-friendly dataset files used for the portfolio repository.

## Included

- `shoe_dataset.csv` — full label table supplied with the project files
- `sample_dataset.csv` — subset tied to committed sample images
- `sample_images/` — compressed representative images from Scene 1 and Scene 2
- `metadata/raw_archive_manifest.json` — summary of raw source archives
- `metadata/sample_image_manifest.csv` — mapping of included sample frames

## Not included

The full raw Scene 1 and Scene 2 archives are not committed because they are hundreds of MB. Keep those externally or use Git LFS if full raw image reproduction is required.
