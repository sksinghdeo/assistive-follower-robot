# Portfolio Scope

This repository is a cleaned, recruiter-facing version of a class/team robotics project. It is organized to show the engineering workflow clearly: data collection, labeling, model training, ROS 2 integration, and hardware-aware navigation.

The raw datasets are not committed in full because they are several hundred MB. The repo includes sample frames, the label CSV, the trained model file, source notes, and scripts needed to reproduce the workflow with the full data stored externally.
